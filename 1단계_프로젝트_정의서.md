# 📊 매일 경제 학습 서비스 — 1단계 프로젝트 정의서

> **한 줄 소개**
> 바쁜 30~50대 직장인·주부가 매일 5~10분씩 오늘의 경제 핵심과 흐름을 소화하고,
> 관련 주식 업종·기업까지 자연스럽게 연결해서 학습하는 서비스

---

## 1. 문제 정의 (Why)

기존 경제 공부의 4가지 어려움:

| # | 문제 | 사용자가 실제로 느끼는 답답함 |
|---|------|---------------------------|
| 1 | 뉴스가 너무 많다 | "뭐부터 봐야 할지 모르겠고, 결국 다 안 본다" |
| 2 | 어려운 용어가 많다 | "금리 인상? 양적긴축? 매번 검색하기 귀찮다" |
| 3 | 어디서부터 시작할지 모른다 | "체계 없이 뉴스만 봐서 지식이 안 쌓인다" |
| 4 | 며칠 하다가 안 본다 | "루틴이 없어서 3일 만에 흐지부지된다" ⭐**(리텐션 이슈, 가장 중요)** |

---

## 2. 타깃 사용자 (Who)

### 메인 페르소나
- **30~50대 직장인** (재테크 관심, 주식 계좌 보유)
- **주부** (생활 밀착 경제 관심, 손실 회피 성향)

### 사용 맥락
- 📱 출퇴근 자투리 시간 (5~10분)
- 🏠 집안일 사이 짬 (주부층, 오디오 재생 니즈)
- 🌙 하루 마무리 시간 (뉴스 정리)

### 핵심 동기
- "돈 벌고 싶으니까 경제를 알아야겠다" (실용 지향)
- "체계적으로 조금씩 쌓아가고 싶다"
- "짧고 명확하게, 부담 없이"

---

## 3. 서비스 컨셉 (What)

### 핵심 방향
> **더 많이 보여주지 않는다. 오늘 꼭 알아야 할 것만, 맥락과 함께.**

### 3대 방향
1. **선택과 집중** — 하루 3개 뉴스, 1개 용어 (더 늘리지 않는다)
2. **맥락 중심** — "무슨 일이 있었나"보다 "왜 중요한가 + 앞으로 어떻게 될까"
3. **꾸준함 설계** — 학습 이력·스트릭·주간 리뷰로 습관 형성

---

## 4. 핵심 기능 (How)

### 🎯 MVP 필수 기능 (v1.0)

#### 1) 오늘의 뉴스 요약 (3개)
- 매일 아침 큐레이션된 핵심 뉴스 **3개**
- 각 뉴스 구조:
  - **30초 요약** (핵심 사실)
  - **왜 중요한가** (배경·맥락)
  - **어떤 영향이 생기나** (예상 시나리오 2~3가지)
  - **관련 업종·종목 태그** (교육 목적, 매수 추천 아님)
- 난이도 배지 (⭐~⭐⭐⭐)
- **오디오 재생** 옵션 (주부·운전자 대응)

#### 2) 매일 배우는 경제 용어 (1일 1개)
- 하루 1개 원칙 (부담 낮추기)
- 3단 구조: **한 줄 정의 → 쉬운 예시 → 오늘 뉴스에서 어떻게 쓰였나**
- 개인 사전 자동 저장

#### 3) 경제 캘린더
- FOMC, CPI, 금통위, 실적 발표 등 **주요 일정**
- D-day 푸시 알림
- 이벤트 사전 설명 + 사후 리포트 자동 발행

#### 4) 나만의 경제 노트
- 학습한 뉴스·용어 **자동 아카이브**
- 개인 하이라이트·코멘트
- 태그·검색 기능
- AI 질문: "지난 주 배운 것 중 금리 관련만 정리해줘"

#### 5) 업종·종목 연결
- 뉴스 → 관련 업종 → **업종 대표 기업 5~10개** 카드로 표시
- 최근 주가 흐름 (교육 관점 그래프)
- 국내, 해외 
- ⚠️ "매수 추천" 표현 절대 금지 (자본시장법 회피)

---

### 🚀 확장 기능 (v2.0 이후)

#### 6) 경제지도 🗺️
- 인포그래픽: **금리 → 환율 → 원자재 → 업종** 인과 관계 시각화
- 오늘의 뉴스가 지도의 어느 지점에서 발생했는지 표시
- 학습이 쌓일수록 지도가 밝아짐 (게이미피케이션)

#### 7) 업종별 대표 기업 지도
- 반도체·2차전지·바이오·금융·유통·에너지 등
- 업종별 대장주·대표주 한눈에
- 업종 간 상관관계 시각화

#### 8) 주간 브리핑 (일요일 저녁 발행) ⭐
- 지난주 학습 복습 + 다음주 주요 일정 예고
- **리텐션 강화 핵심 기능** (일요일 앱 재방문 유도)

#### 9) 학습 스트릭·뱃지
- 연속 학습일 카운트
- 마일스톤 뱃지 (7일·30일·100일)
- **"며칠 하다가 안 본다" pain point 대응**

#### 10) 온보딩 진단
- 초기 3~5문항으로 **레벨(초보/중급/고급)·관심 업종** 파악
- 개인화된 콘텐츠 큐레이션 시작점

#### 11) AI Q&A
- "오늘 뉴스 중 이 부분이 이해가 안 돼요"
- 사용자 학습 이력 기반 개인화 답변

#### 12) 인사이트 공유 카드
- 오늘의 학습 내용을 **예쁜 카드**로 SNS 공유
- 자연 발생적 마케팅 채널

## 5. 차별화 전략 (vs 경쟁)

| 경쟁 서비스 | 우리와의 차별점 |
|-----------|--------------|
| 삼프로TV·언더스탠딩 | 훨씬 짧고(5~10분), 텍스트·카드 기반, 시간 부담 없음 |
| 매경·한경 뉴스레터 | 상호작용 있음(퀴즈·노트), 종목 연결, 개인화 |
| 종목토론방·오픈채팅 | 학습 중심, 노이즈 없음, 신뢰성 확보 |
| 토스·키움 등 증권사 앱 | 매매가 아닌 **학습**이 메인, 초보 친화적 |
| 챗GPT 등 범용 AI | 매일 큐레이션·습관 형성 구조, 오늘의 핵심을 알아서 정리 |

### 우리만의 무기 (Moat)
1. **"교육 + 종목 매칭" 절묘한 경계 포지셔닝** — 규제 회피 & 실용성 동시 확보
2. **경제지도** — 시각적 자산, 후발주자가 따라오기 어려움
3. **개인 학습 아카이브** — 사용자가 축적할수록 이탈이 어려워지는 lock-in

## 6. 기술 스택 고려사항

### 데이터 소스
- **뉴스 수집**: 연합뉴스·매경·한경 등 RSS/API (⚠️ 저작권 확인 필수, 원문 재배포 금지)
- **주식 데이터**: KRX 오픈API, 한국투자증권 오픈API, KIS Developers
- **경제 지표**: 한국은행 ECOS API, 통계청 KOSIS API
- **글로벌 지표**: FRED API (미 연준), Alpha Vantage 등

### AI/자동화
- **뉴스 요약·해설**: LLM API (Claude / OpenAI GPT / Gemini)
- **개인화 큐레이션**: 사용자 행동 로그 기반 추천
- **AI Q&A**: RAG (개인 노트 벡터 검색 + LLM 응답)

### 앱 스택 (제안)
- **Frontend**: React Native (iOS·Android 동시 개발, 학습 곡선 완만)
- **Backend**: Node.js + PostgreSQL, Redis (캐싱)
- **AI 파이프라인**: Python + LangChain (뉴스 요약 워크플로우)

### ⚠️ 법률·규제 리스크
| 이슈 | 대응 방안 |
|-----|---------|
| 자본시장법(투자자문업 등록) | **"교육"** 포지셔닝 유지, "매수 추천" 표현 절대 금지, 하단 면책 고지 |
| 뉴스 저작권 | 요약·인용은 공정이용 범위, 원문 링크로 이동 유도 |
| 개인정보(학습 노트) | 개인정보처리방침 명확화, 암호화 저장 |

## 7. 수익 모델 (Monetization) — 참고 제안

- **Freemium**: 기본 콘텐츠 무료, 심화 콘텐츠·경제지도·AI Q&A 유료
- **월 구독**: 4,900~9,900원 (커피 한 잔 프레이밍)
- **B2B**: 증권사·은행에 화이트라벨 제공 가능성
- **광고 배제** (교육 서비스 신뢰도 유지 위해)
